# Watermarker-Air

本插件基于Sigil插件机制开发，可为EPUB电子书中的图片批量添加水印，并支持通过黑名单规则排除无需处理的图片。插件提供图形化配置界面，方便用户自定义黑名单规则，并对待转换文件进行二次确认。

## 🌟 核心特性

- **多格式兼容**：支持读取并处理 JPG、JPEG、PNG 以及 WebP 格式的图片。
- **矢量与位图水印源**：不仅支持常规的光栅图片作为水印，还支持解析 SVG 文件（通过底层 `resvg-py` 库高质量渲染为图形缓冲区）。
- **上下文智能过滤**：具备强大的黑名单防御机制。通过继承 `HTMLParser` 追踪标签堆栈，插件能够自动跳过特定 HTML 文件（如 `cover.xhtml`）、特定图像名称、带有特定 CSS 类名（如 `no-watermark`）的容器，以及特定 HTML 标签内的图片。
- **双阶段图形用户界面 (GUI)**：
  - **章节选择器**：第一阶段自动列出所有内部 HTML 文件，并会智能预选当前您在 Sigil 主界面中正在编辑的章节。
  - **图像二次确认**：提供可选的二次确认选项。启用后，系统会扫描选定章节，并在第二阶段直观展示所有命中的图片，允许您进行精确的单图剔除。
- **多线程并发加速**：底层采用 `ThreadPoolExecutor` 构建并发任务池，默认开启 4 线程渲染，可成倍缩短包含海量图片的 EPUB 项目的处理时间。
- **体积优化与量化**：内置智能压缩策略。在指定特定输出质量时，会自动调用 `imagequant` 库对 PNG 图像执行颜色量化压缩，有效避免加水印导致的电子书体积膨胀。

## 📦 环境依赖

在运行此插件之前，请确保您的 Sigil 外部 Python 环境或插件自带的 `vendor` 目录中安装了以下第三方依赖库：

- `Pillow` (核心图像处理引擎)
- `pyyaml` (配置文件解析)
- `resvg-py` (SVG 矢量文件光栅化渲染)
- `imagequant` (PNG 高级有损压缩优化)
- `cffi` (为Python程序提供接口)

## 🚀 安装指南

1. 在 Sigil 菜单栏`插件` → `编辑插件`中添加插件。
2. 安装成功后插件位于 Sigil 的官方插件目录中。对于 Windows 系统，该默认路径通常位于 `…\sigil-ebook\sigil\plugins`。
3. 重新启动 Sigil，您即可在顶部的`插件`菜单中看到并执行此工具。

## ⚙️ 配置文件说明

插件在首次初始化时，如果未检测到配置，会自动在当前目录生成默认的 `watermarker_config.yaml` 文件。您可以根据项目需求自由定制以下参数：

- **水印属性**：配置 `image_filename` 设定水印源文件；使用 `width` 设定水印大小（支持固定像素或相对原图的百分比）；调节 `opacity` 设置透明度混合级别；以及配置 `rotation` 改变水印倾斜角度。
- **定位边距**：通过 X 轴和 Y 轴的 `margins` 参数（同样支持像素或百分比），您可以将水印精准锚定在图片的任意角落。
- **质量控制**：通过 `0–100` 的数值控制输出渲染质量。
- **过滤规则列表**：在配置文件中声明您希望全局跳过处理的特定 CSS 类名或文件标识符。

## 💡 使用流程

1. 在 Sigil 中打开您需要处理的 EPUB 电子书。
2. 找到`插件`菜单并启动 **Watermarker-Air** 插件。
3. 在弹出的第一阶段 GUI 窗口中，勾选需要批量注入水印的章节。
4. 如果您勾选了「二次确认」，请在随即弹出的第二阶段窗口中，审阅并微调实际将要处理的图像列表。
5. 点击执行后，插件将在后台开始多线程渲染。完成后，您可以打开 Sigil 的 **插件控制台** 查看详细的运行报告，报告会列出所有成功合成的图像记录以及触发过滤规则被跳过的图像记录。
